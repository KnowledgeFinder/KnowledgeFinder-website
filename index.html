What is KnowledgeFinder?

KnowledgeFinder is an Open Source knowledge management and visualization environment. It is used by scientists to manage their documents and data and to improve their joint work.
If you're interested in contributing to KnowledgeFinder please don't hesitate and contact us. The source code can be found on GitHub.
Sign up on our mailing list for latest news, to ask questions and to get in touch.
